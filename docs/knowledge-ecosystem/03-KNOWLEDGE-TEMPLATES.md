# KNW-KE-ARCH-003 — Knowledge Templates

**Phase:** 3.0C.5  
**Status:** CANONICAL  
**Owner:** architecture-board  
**Version:** 1.0.0

---

## Purpose

Every KnowledgeObjectType has exactly one canonical template. Templates enforce the Universal Schema (16 sections), required fields, and validation rules. Authors copy a template and fill in the blanks.

---

## Template Format

```yaml
# templates/{type_lower}-template.yaml
_template_version: "1.0.0"
_object_type: "{TYPE}"
_required: [field1, field2]   # fields that must be non-empty
_optional: [field3]           # fields with defaults

# --- Universal Schema (16 sections, all required) ---

knowledge_id: "KNW-{DOMAIN}-{TYPE}-{NNN}"   # fill in
object_type: {type_lower}
name: ""             # REQUIRED: Title Case, max 80 chars
version: "1.0.0"
status: DRAFT
owner: ""            # REQUIRED: "team:name" or "user:name"
description: ""      # REQUIRED: min 20 chars, imperative mood

classification:
  domain: {DOMAIN}
  layer: 0
  category: ""

identity:
  global_id: ""      # auto-generated by registry
  canonical_name: "" # auto-generated: {ns}.{type}.{slug}
  knowledge_uri: ""  # auto-generated: knw://...

tags: []             # REQUIRED: min 1 tag, lowercase, hyphenated

evidence:
  items: []
  evidence_score: 0.0

quality:
  overall_score: null
  computed_at: null

confidence:
  declared: 0.5
  composite: null

traceability:
  satisfies: []      # knowledge_id of requirements this satisfies
  implements: []
  tests: []

lifecycle:
  review_status: PENDING
  approved_by: ""
  approved_at: null

relationships: []    # populated by relationship engine

metadata:
  created_by: ""
  created_at: ""
  source_doc: ""

# --- Type-specific fields below ---
```

## Template Validation Rules

| Rule | Description |
|------|-------------|
| TV-001 | `knowledge_id` must match regex `^KNW-[A-Z][A-Z0-9]*-[A-Z][A-Z0-9]*-[A-Za-z0-9_-]+$` |
| TV-002 | `name` must be non-empty, ≤ 80 chars, no leading/trailing whitespace |
| TV-003 | `description` must be ≥ 20 chars |
| TV-004 | `owner` must match `team:*` or `user:*` or `system:*` |
| TV-005 | `tags` must have at least 1 entry; each tag lowercase and hyphen-separated |
| TV-006 | `version` must be valid semver `MAJOR.MINOR.PATCH` |
| TV-007 | `confidence.declared` must be in [0.0, 1.0] |
| TV-008 | All type-specific `_required` fields must be non-empty |
