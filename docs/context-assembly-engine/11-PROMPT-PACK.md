# CAE-DOC-011 — Prompt Pack

**Phase:** 3.0D.1
**Status:** CANONICAL
**Owner:** architecture-board
**Version:** 1.0.0

---

## Purpose

The Prompt Pack is the output format for AI agent consumers. It takes the raw
context pack from the assembler and formats it into a structured block suitable
for injection into an LLM prompt, system prompt, or RAG pipeline.

PromptPack is the default output when `consumer_profile.type == AI_AGENT`.

---

## Prompt Pack Schema

```yaml
prompt_pack:
  pack_id: string                      # UUID, unique per assembly
  pack_type: PROMPT
  query_id: string
  query_text: string
  intent_type: string
  generated_at: string                 # ISO 8601

  preamble: string                     # generated by assembler

  prerequisites:                       # optional — present only if needed
    - knowledge_id: string
      name: string
      content: string                  # at L2 or L3

  primary:
    knowledge_id: string
    canonical_name: string
    compression_level: string          # L3 | L4 | L5
    content: string                    # rendered KIL block
    token_count: integer

  context_objects:
    - knowledge_id: string
      role: string                     # CONTEXT | PREREQUISITE | EVIDENCE
      compression_level: string
      content: string
      relevance_score: float
      token_count: integer

  anti_confusion:                      # objects with high similarity to primary
    - knowledge_id: string
      name: string
      micro: string                    # L2 — name + purpose only
      scope_clarification: string

  guard_block:
    scope_boundaries: [string]         # what this object IS NOT
    hedges: [string]                   # where primary knowledge may be incomplete
    typical_mistakes: [string]         # common errors from cortex.common_errors
    confidence_hedge: string           # from confidence.level hedging_rule

  reasoning_scaffold:
    chain_of_thought: string | null    # from thinking.chain_of_thought_template
    conclusion_format: string | null   # expected output format hint

  pack_metadata:
    total_tokens: integer
    budget_declared: integer
    budget_utilization: float          # total_tokens / budget_declared
    context_confidence: float          # min(obj.confidence.overall_score for all objects)
    assembly_latency_ms: integer
    objects_included: integer
    objects_dropped: integer
    airs_min: float                    # min AIRS of all included objects
```

## PromptPack Build Algorithm

```
BUILD_PROMPT_PACK(raw_pack, intent_record, consumer_profile):

  # Validation: all objects must have AIRS >= 0.60 (CI-08)
  FOR obj in raw_pack.objects:
    IF obj.ai_readiness_score < 0.60:
      RAISE AssemblyError(AIRS_TOO_LOW)
      message: "{obj.knowledge_id} AIRS={obj.airs} below 0.60 minimum"

  # Build sections
  pack = PromptPack(
    pack_id = uuid4(),
    pack_type = PROMPT,
    query_id = intent_record.query_id,
    query_text = intent_record.raw_query,
    intent_type = intent_record.intent_type,
    generated_at = now()
  )

  # Preamble (always present)
  pack.preamble = raw_pack.preamble

  # Prerequisites section
  pack.prerequisites = [
    format_object(obj) for obj in raw_pack.sections
    WHERE section_type == PREREQUISITE
  ]

  # Primary (always present)
  primary_obj = raw_pack.get_section(PRIMARY).objects[0]
  pack.primary = {
    knowledge_id: primary_obj.knowledge_id,
    canonical_name: primary_obj.index_entry.canonical_name,
    compression_level: primary_obj.compression_level,
    content: primary_obj.content,
    token_count: primary_obj.token_count
  }

  # Context objects
  pack.context_objects = [
    format_context_object(obj)
    for obj in raw_pack.sections
    WHERE section_type in {CONTEXT, EVIDENCE}
  ]

  # Anti-confusion
  pack.anti_confusion = [
    format_anti_confusion(obj)
    for obj in raw_pack.sections
    WHERE section_type == ANTI_CONFUSION
  ]

  # Guard block (built from primary object's KIL fields)
  primary_kil = INDEX.get(primary_obj.knowledge_id)
  pack.guard_block = {
    scope_boundaries: primary_kil.cortex.why_not + primary_kil.semantic.scope_boundaries,
    hedges: derive_hedges(primary_kil, pack.context_objects),
    typical_mistakes: primary_kil.cortex.common_errors[:3],
    confidence_hedge: primary_kil.confidence.hedging_rule_for_level()
  }

  # Reasoning scaffold (for REASONING and CONTEXT_PRODUCTION intents)
  IF intent_record.intent_type in {REASONING_QUERY, CONTEXT_PRODUCTION}:
    pack.reasoning_scaffold = {
      chain_of_thought: primary_kil.thinking.chain_of_thought_template,
      conclusion_format: intent_record.expected_output_format
    }

  # Metadata
  pack.pack_metadata = compute_metadata(raw_pack, pack)

  RETURN pack
```

## Guard Block Hedge Derivation

```
derive_hedges(primary_kil, context_objects):

  hedges = []

  # Primary confidence hedge
  IF primary_kil.confidence.overall_score < 0.90:
    hedges.append(primary_kil.confidence.hedging_rule)

  # Context confidence hedges (for CONFIDENT level or below)
  FOR obj in context_objects:
    IF obj.index_entry.confidence.overall_score < 0.80:
      hedges.append(
        f"Note: {obj.knowledge_id} has confidence {obj.confidence.level} "
        f"({obj.confidence.overall_score:.2f}) — verify before relying on it."
      )

  # Missing required context
  required = primary_kil.ai_context.related_objects
               WHERE fetch_priority == ALWAYS
  included_ids = {obj.knowledge_id for obj in context_objects}
  missing = [r for r in required WHERE r.knowledge_id not in included_ids]
  FOR m in missing:
    hedges.append(f"Required context missing: {m.knowledge_id} ({m.relationship}) "
                  f"— budget may be insufficient for complete context.")

  RETURN hedges
```

## Rendered Format

PromptPack is serialized to a structured YAML block or Markdown block for injection:

```
[CONTEXT FOR: {query_text}]
[INTENT: {intent_type}]
[PRIMARY: {canonical_name} ({knowledge_id})]
[CONFIDENCE: {context_confidence:.2f}]

{preamble}

=== PREREQUISITES ===
{prerequisites rendered}

=== PRIMARY OBJECT ===
{primary.content}

=== CONTEXT ===
{context_objects rendered, each labeled with role and knowledge_id}

=== ANTI-CONFUSION ===
Do NOT confuse the above with:
{anti_confusion rendered, each with scope_clarification}

=== GUARD ===
SCOPE BOUNDARIES:
{scope_boundaries bulleted}

HEDGES:
{hedges bulleted}

WATCH OUT FOR:
{typical_mistakes bulleted}

{confidence_hedge}

=== REASONING ===
{chain_of_thought if present}
Expected output: {conclusion_format if present}
```

## Rules

| Rule | Statement |
|------|-----------|
| CAE-046 | PromptPack must not be emitted if any object has AIRS < 0.60 |
| CAE-047 | Guard block is mandatory in every PromptPack — it cannot be suppressed |
| CAE-048 | `pack_id` must be a new UUID per assembly — reuse of pack_id is forbidden |
| CAE-049 | `context_confidence` is the MINIMUM, not average, of all included objects |
| CAE-050 | Anti-confusion section must use the label "Do NOT confuse" verbatim |
